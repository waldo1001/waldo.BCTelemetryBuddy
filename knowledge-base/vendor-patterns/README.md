# Vendor Patterns

This folder contains BC Telemetry knowledge articles specific to common third-party integrations and ISV solutions (e.g., CRM connectors, e-commerce platforms, EDI systems).

Add articles here when you have telemetry patterns that are specific to a vendor integration's behavior — lock patterns, error codes, performance characteristics, etc.

See the root [README.md](../README.md) for the article format.
